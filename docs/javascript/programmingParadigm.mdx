# JavaScript编程范式

**什么是JavaScript**

:::info
javascript是一种基于对象和事件驱动的轻量级编程语言
:::

**javascrip**t是一种**嵌入型(embedded)**语言，它本身提供的核心语法不多

**JavaScript**本身不提供任何与**I/O**有关的**API**，主要依靠宿主环境提供，所以javascrpt适合嵌入大型的应用程序环境，去调用宿主环境提供的底层API。

>:building_construction:例如在浏览器中，网络I/O，完全依赖于浏览器。在NodeJs中则依赖于底层的libuv库

目前常见的宿主宿主环境就是浏览器，例如`chrome、edge、safari、firefox`。还有运行在服务端的NodeJs

从语法的角度看，**javascript**是一种“对象模型”的语言。各种宿主环境通过这个模型，描述自己的功能和操作接口，从而通过**javascript**控制这些功能。但是**javascript**并不是完全的面向对象的语言，它是基于原型链的方式实现的。

实际上，**javascript**身上有着多门语言的血统：

* 借鉴`C`语言的基本语法（条件分支，循环等）
* 借鉴`Java`的数据类型和内存管理（堆、栈）
* 借鉴`Scheme`语言，将函数提升到“一等公民”的地位（高阶函数）
* 借鉴`Self`语言，使用基于原型的继承机制（prototype）

:rocket:虽然是各家特性杂糅，但如文章开头所说，虽然是各家特性杂糅，但如文章开头所说，**JavaScript**是一门轻量级的语言，其核心功能是相当精简的，只包括两部分：

* 基本的语法构造（操作符、控制结构、语句）
* 标准库（如`Date、Math、Array`等一系列对象提供的方法）

## 面向对象编程

面向对象很重要的一方面，就是类的继承。javascript类继承基于prototype

下面是一个最简单的继承例子：

```js
function Father() {
    this.greet = function() {
        console.log('hallo')
    } 
}
function Son(params) {}

Son.prototype =  new Father()

new Son.green() //=> hallo
```
